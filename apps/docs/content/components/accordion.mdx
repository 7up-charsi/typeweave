### item

Contains all the parts of a collapsible section. by default trigger wraps button in h3 heading but you can change heading level with asChild

<Props
  data={[
    {
      name: 'value',
      type: 'string',
      required: true,
      description: <>A unique value for the item.</>,
    },
    {
      name: 'disabled',
      type: 'boolean',
      description: (
        <>
          When true, prevents the user from interacting with the item.
        </>
      ),
    },
  ]}
/>

### header

by default renders <code>h3</code> element. it must be nested inside Accordion.Item

<Props data={[{ name: 'asChild' }]} />

### trigger

Toggles the collapsed state of its associated item. it must be nested inside Accordion.Item

<Props data={[{ name: 'asChild' }]} />

### content

Contains the collapsible content for an item and it does not accept any props other than <code>React.HTMLAttributes</code>.

### arrow

it does not accept any props other than <code>React.SVGProps</code>. it must be nested inside Accordion.Trigger
. By default it renders a svg chevron and if you want to change this icon and animate it on expand/collapse then [see example](#custom-indicator-example)
