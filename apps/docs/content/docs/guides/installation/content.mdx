## configure Tailwind css

Webbo-ui is built on top of Tailwind CSS, so you need to install Tailwind CSS first. You can follow the official installation guide to install Tailwind CSS. Then follow the steps below.

<Steps>
### Install theme package
To install, run following command in your terminal

```bash
pnpm add @webbo-ui/theme
```

### add theme plugin

then configure it in your <code data-highlight>tailwind.config.ts</code>, leave `content` array as is we will modify it later.

```ts
import { WebboUi } from '@webbo-ui/theme';

/** @type {import('tailwindcss').Config} */
export default {
  content: [
    // your stuff...
  ],
  plugins: [WebboUi()],
};
```

</Steps>

## component installation

Webbo-ui provides each component as individual package. Install the only components you need and don't forgot to add <code data-highlight>style paths</code> in <code data-highlight>tailwind.config.ts</code> as these paths are slightly different. **These** <code data-highlight>style paths</code> **are given at every compnent page for you so you can copy and paste them**.

Asume you have configured tailwindcss as above then follow the steps below.

<Steps>
### install component

Now, let's install the component you want to use. For example, if you want to use the Button component, you need to run this command in your terminal:

```bash
pnpm add @webbo-ui/button
```

### tailwind css styles

Button component has installed. Now add its <code data-highlight>style paths</code> in your `content` array in <code data-highlight>tailwind.config.ts</code> for button styles.

```txt
'./node_modules/@webbo-ui/theme/dist/components/button.js',
'./node_modules/@webbo-ui/theme/dist/classes.js',
```

### import component

then you can import and use componet.

```tsx
import { Button } from '@webbo-ui/react'

export default funtion App() {
  return <Button>webbo-ui</Button>
}
```

</Steps>
